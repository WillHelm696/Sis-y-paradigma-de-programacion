Para crear un párrafo en HTML, usa la etiqueta < p > (del inglés paragraph) abriendo con < p > y cerrando con < /p >. Todo el texto que escribas entre estas dos etiquetas se considerará un párrafo y el navegador lo mostrará como un bloque de texto separado.

Subtitulo del articulo h2

Etiqueta de apertura: Comienza el párrafo con < p >.
Contenido: Escribe el texto que quieres que aparezca en el párrafo.
Etiqueta de cierre: Finaliza el párrafo con < /p >.
Separación: Los párrafos creados de esta forma se muestran con un margen inferior y superior por defecto, separándose del texto circundante.

Etiqueta < blockquote >

La etiqueta < blockquote > en HTML se usa para representar una sección que ha sido citada de otra fuente, como un artículo o un blog. Generalmente, los navegadores muestran el texto con sangría para distinguirlo del contenido circundante, ya que es un elemento de bloque que crea un salto de línea antes y después. Se puede utilizar el atributo cite para especificar la URL de la fuente original de la cita

Etiqueta < cite >

La etiqueta < cite > en HTML se utiliza para identificar y definir el título de una obra creativa, como un libro, una película, una canción o una pintura. Esta etiqueta es semántica, lo que significa que proporciona información a los motores de búsqueda y otras tecnologías de asistencia, y generalmente hace que el texto aparezca en cursiva en el navegador.
La etiqueta < i > permite cambiar el estilo de letra a otro

Etiquetas de listas

    Etiquetas de formularios

    Atributos principales

    action: Especifica la URL del script del servidor que procesará los datos del formulario.
    method: Define el método HTTP para enviar el formulario. Los más comunes son:

    • GET: Los datos se envían como parte de la URL.
      POST: Los datos se envían en el cuerpo de la solicitud.
    • enctype: Indica cómo deben codificarse los datos del formulario al enviarse. Es crucial para subir archivos, donde se usa multipart/form-data.

    Otros atributos importantes

    target: Define dónde se mostrará el resultado de la respuesta del formulario. _blank lo abrirá en una nueva pestaña.
    accept-charset: Especifica la codificación de caracteres para el formulario. Se usa para asegurar la correcta interpretación de caracteres de diferentes idiomas.
    autocomplete: Permite que el navegador sugiera valores guardados previamente en los campos del formulario, mejorando la experiencia del usuario.
    novalidate: Un atributo booleano que desactiva la validación del formulario antes de enviarlo.

    La etiqueta input y lavel

    La etiqueta < label > se usa para asociar texto descriptivo con un campo de entrada (< input >) mediante los atributos for (en < label >) en id ( < input >) . Los atributos comunes de < input > incluyen type para especificar el tipo de dato (como text, number, email), value para el valor inicial y placeholder para un texto de ejemplo.
    Atributos de la etiqueta < label > for: Vincula la etiqueta con un elemento de formulario específico. El valor debe coincidir con el atributo id del < input >. form: Asocia la etiqueta con un formulario específico si el label está fuera de un form.
